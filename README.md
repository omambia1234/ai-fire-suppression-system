# AI Fire Suppression System

An AI-powered real-time fire detection system using OpenCV and computer vision.

## Features
- Real-time camera fire detection
- Snapshot logging
- Event tracking system (future: PostgreSQL integration)
- Alert system (in progress)

## Tech Stack
- Python
- OpenCV
- NumPy
- PostgreSQL (planned)
- Metabase (analytics layer)

## Status
Work in progress system transitioning into production-grade AI monitoring tool.
